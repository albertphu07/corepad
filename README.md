## ✨ Features

- ⌨️ Fully programmable keys
- ⚡ Powered by the Seeed Studio XIAO RP2040
- 🐍 KMK Firmware support
- 🔌 USB-C connectivity
- 💡 RGB lighting support
- 🖥️ OLED display support
- 🧩 Open-source hardware and firmware
- 🛠️ Easy to modify and customize
